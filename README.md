# ai-multifeature-app


An AI-powered full-stack application** built with **Spring Boot (Java)** and React.js, featuring interactive AI chat, image generation, and recipe creation using OpenAI integration 



Features

AI Chat – Chat with an intelligent AI assistant  
AI Image Generator – Generate creative images from text prompts  
AI Recipe Creator – Get recipes based on ingredients or meal type  
Full Stack Integration – Java Spring Boot backend with React frontend  
CORS Configured – Seamless communication between frontend and backend
